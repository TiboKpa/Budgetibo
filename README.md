# Budgetibo 2.0

Application de gestion de budget personnel et de patrimoine, inspirée par la méthode 45-10-10-35 et le design Finary.

## Architecture
- **Backend**: Node.js, Express, SQLite (better-sqlite3)
- **Frontend**: React, Vite, TailwindCSS, Recharts

## Fonctionnalités
- Vue mensuelle détaillée avec répartition budgétaire configurable
- Gestion des dépenses fixes et variables avec sous-catégories libres
- Clôture de mois et réconciliation de l'épargne
- Vue Bilan annuel avec graphiques d'évolution
- Import/Export de données

## Installation

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
